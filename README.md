# Kigali Tourism Website

## Description
This is a comprehensive website showcasing tourism destinations, cultural experiences, and dining options in Kigali, Rwanda. The project includes 5 pages with HTML5 and CSS3 features, providing visitors with valuable information about attractions, dining, and cultural experiences.

## Project Pages

### 1. **Home Page** (index.html)
- Hero section with background image
- Featured attractions cards (Mount Kigali, Restaurants)
- About Kigali section
- Contact form for inquiries
- Navigation to all destination pages

### 2. **Kigali Genocide Memorial** (memorial.html)
- Comprehensive information about the memorial
- Visitor information table (hours, fees, duration)
- Photo gallery
- Embedded Google Maps
- Tips and guidelines for visitors
- External link to official website

### 3. **Nyamirambo Walking Tour** (nyamirambo.html)
- Cultural tour introduction
- **Ordered list** - Step-by-step tour guide (8 steps)
- Photo gallery
- Tour pricing cards
- Tips for visitors

### 4. **Mount Kigali Hiking** (mount.html)
- Hiking information cards (duration, difficulty, distance)
- **Unordered lists** - Packing checklist organized by categories
- Photo gallery
- Embedded **video** of hiking experience
- Tips for safe hiking

### 5. **Restaurants** (restaurants.html)
- Featured restaurants (Heaven, Pili Pili, Umami)
- **Table** with menu items and prices
- **Audio guide** about Rwandan cuisine
- Photo gallery
- **Reservation form** for booking tables

## How to Use

1. **View the website**: Open `index.html` in a web browser
2. **Navigate**: Use the navigation bar to visit different pages
3. **Explore**: Click on links and view all content including images, tables, and forms
4. **Contact**: Fill out the contact form on the home page
5. **Book**: Use the reservation form on the restaurants page
